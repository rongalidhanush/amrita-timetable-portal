# 📅 Timetable Portal

A modern timetable management application built with **Next.js**. It helps students and faculty view and manage academic schedules through a clean and responsive interface.

## 🌐 Live Demo

🔗 **https://timetable-portal.vercel.app/**

---

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev

# or
yarn dev

# or
pnpm dev

# or
bun dev
```

Open **http://localhost:3000** in your browser to see the application.

You can start editing the page by modifying:

```text
app/page.tsx
```

The page will automatically update as you save your changes.
